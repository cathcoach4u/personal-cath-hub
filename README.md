# Cath Hub

Personal health & household management PWA built with vanilla HTML/CSS/JS and Supabase.

**Live:** https://cathcoach4u.github.io/personal-cath-hub/

## Features

- 📱 Progressive Web App (installable on phones)
- 🔐 Supabase authentication with PIN lock
- 🏥 Medical tracking (medications, sessions, mental health)
- 🏋️ Workout logging
- ✅ To-do lists with categories and due dates
- 📅 Daily habit tracking with streaks
- 🛒 Shopping management
- ✈️ Travel planning
- 🌙 Dark mode support
- 📱 Responsive design (desktop sidebar, mobile hamburger menu)

## Screens

- **Dashboard** - Overview with daily habits, mental health stats, to-do summary
- **To-Do** - Personal tasks with categories and due dates
- **Workout Log** - Exercise tracking with sets/reps
- **Medical** - Medications, mental health plans, Medicare tracking
- **My Habits** - Daily habit tracking with 7-day view
- **Travel** - Travel planning
- **About** - App info and features

## Tech Stack

- Vanilla HTML/CSS/JavaScript (no frameworks)
- Supabase for backend & authentication
- GitHub Pages for hosting
- PWA manifest for app installation
- No build step required

## Development

All files are in the root directory. Simply edit HTML files and push to main - GitHub Actions will deploy automatically.

### Key Files

- `index.html` - Main app (all screens)
- `config.js` - Centralized configuration
- `morning.html` - Morning medication auto-logging
- `todo.html` - Quick to-do list
- `workout.html` - Workout logging
- `.github/workflows/pages.yml` - Auto-deployment to GitHub Pages
