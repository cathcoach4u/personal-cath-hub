💠 Vestibular Migraine Care Plan

1. Acute Treatment (During an Attack)
FOR DIZZINESS
• Prochlorperazine 5–10mg up to 3x daily as needed
• If ineffective: Cizinate (cinnarizine/dimenhydrinate) 20/40mg up to 3x daily
FOR NAUSEA
• Ondansetron 8mg wafers up to 3x daily as needed
Side effects: Drowsiness, nausea, headache, dry mouth. Rare Parkinson-like symptoms if Cizinate taken daily for 4+ weeks.
2. Migraine Prevention
FIRST LINE (PREFERRED)
• Magnesium: 400–600mg daily
• Riboflavin (Vitamin B2): 400mg daily
• Optional: CoQ10 150–300mg daily
IF SYMPTOMS CONTINUE
• Amitriptyline: Start 10mg nightly, increase to 20mg if needed
Side effects: Drowsiness, dry mouth, dry eyes, weight gain.
3. Weekly Migraine Check-In (10 min)
Sleep: Regular bedtime, screens off 1hr before, caffeine cut-off 3pm
Diet: Regular meals, note trigger foods
Hydration: 1.5–2L daily, anchors: morning, mid-morning, mid-afternoon
Caffeine: Confirm max daily, reduce gradually if needed
Exercise: 3–4 sessions/week, 30–40 min (walking works)
Ergonomics: Screen breaks, reduce glare, avoid strong scents
Relaxation: 3 sessions/week, 5–10 min (meditation, yoga, breathing)
4. Daily Baseline (2 min)
• Keep wake time consistent
• Do not skip meals
• Start hydration early
• Move your body (minimum 10 min)
• 2–5 min relaxation
5. Migraine-Day Plan
• Hydrate with small sips regularly
• Eat small, steady snacks
• Skip exercise if it worsens symptoms
• Dim lights, limit screens, reduce noise, avoid strong scents
• Slow breathing, short grounding exercises

Stack

• GitHub Pages for hosting
• Claude Code for AI-assisted development
• PWA (installable, standalone mode)
Features
• Dashboard with quick stats, navigation cards, & morning habits link
• To-Do lists with categories & due dates
• AisleMate shopping system (archived — now in Baker Hub)
• Workout logging with exercise library & custom exercises
• Medical records (daily & weekly medications, sessions, contacts, mental health)
• Daily habit tracking (Vyvanse, Valtrex, Levothyroxine) with 7-day streaks
• Daily medication management with morning & evening tracking
• Travel planning
• Home management with zone cleaning cards
• AI assistant with persistent memory (floating chat via Anthropic API, facts & preferences saved to Supabase)
• Dark mode
• PIN lock for quick re-entry
• Responsive design (desktop sidebar, mobile hamburger menu)
